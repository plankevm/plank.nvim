# plank.nvim

Plugin that adds file type & syntax highlighting support for the
[plank](https://plankevm.github.io) language to neovim.

## Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
  "plankevm/plank.nvim",
  dependencies = { "nvim-treesitter/nvim-treesitter" },
}
```
